# AI YouTube Automation v2

This project is an AI automation workflow built using n8n, OpenAI, and Google Sheets.
AI automation workflow built using n8n, OpenAI, and Google Sheets to generate viral YouTube hooks and titles from video transcripts.
## Features
- Takes video transcript input
- Generates viral YouTube hooks using AI
- Saves generated hooks into Google Sheets
- Fully automated workflow

## Tech Stack
- n8n
- OpenAI API
- Google Sheets API

## Workflow
1. Input transcript
2. AI generates hooks
3. Results stored in Google Sheets
