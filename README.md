# AI YouTube Automation Workflow

An AI-powered automation workflow built using n8n, OpenAI, and Google Sheets.

## Features

- AI-generated YouTube hooks
- Viral title generation
- SEO-friendly video descriptions
- Automated Google Sheets integration
- Multi-stage workflow automation

## Workflow Architecture

1. Input transcript
2. Generate hooks using OpenAI
3. Generate titles
4. Generate descriptions
5. Store outputs in Google Sheets

## Tech Stack

- n8n
- OpenAI API
- Google Sheets API
